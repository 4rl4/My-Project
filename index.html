What I learned

A programming language is a communication system between humans and computers that is used to give commands to the computer to perform certain tasks. Some examples of popular programming languages ​​include: Python, Java, C, PHP, Visual basic, HTML, ActionScript, JavaScript, C#, R.

programming language

A database is a collection of data organized and stored electronically, which can contain various types of data such as words, numbers, images, videos, and files. Databases are usually controlled by a database management system (DBMS). Databases have several goals, including: Speed ​​and convenience, Storage space efficiency, Accuracy, Security, Maintaining data harmony.

Database

Web programming or web programming is the process of communicating commands to a computer to build a website-based program so that it can be accessed using the internet The most commonly used programming languages ​​for creating web pages include HTML, CSS, and JavaScript. HTML is used for the basic structure and appearance of websites. CSS is used to regulate the visual appearance of a website. JavaScript is used to make web pages more interactive
